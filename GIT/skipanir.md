## Búum til git repo
1. Búa til möppu sem geymir repo "mkdir verk1" og förum í það "cd verk1"
2. Búum til repo "git init"
   Gerið þessa skipun ls -la þá ætti þetta að birtast
   
   drwxr-xr-x 1 ebe 1049089 0 Dec  8 14:52 .git
